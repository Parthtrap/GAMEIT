## Deployment

To deploy GAMEIT, run:

```bash
  git clone https://github.com/Parthtrap/GAMEIT.git
  cd GAMEIT
  cd client
  npm i
  npm start
  cd ../server
  npm i
  npm start
```
## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express, MongoDB
